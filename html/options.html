🔥 Hot Bookmark Bar 🔥

This extension doesn't actually have any options, but this page can tell you about the extension!

This extension is developed by Nate Dimick and is an open-source project. Got an idea for a feature? Tired of waiting for a feature and want to make it yourself? Here's the Source Code, issues and pull requests are welcome!

Upcoming Features*

  • Chrome Sync for consistancy across multiple devices
  • Recommended Bookmarks based on browsing habits
  • Better Icons
  • Optimized performance
  • Tracking for Other Bookmarks and Mobile Bookmarks as well

* no promised date for upcoming features or guarantee that upcoming features will be added
